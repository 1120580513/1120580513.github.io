## 查看

* **docker --version**：*检查版本*
* **docker --info**：*查看信息*
* **docker image ls --all**：*查看镜象*
* **docker rmi [id]**：*删除镜象*
* **docker container ls --all**：*列出所有容器*
* **docker container stop [id]**：*停止容器*
* **docker container rm [id]**：*删除容器*
* **docker build -t friendlyhello .**：*构建docker*
